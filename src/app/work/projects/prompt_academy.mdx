## Overview

The Prompt Academy is a SaaS learning platform I designed, built and launched on my own. It teaches developers and entrepreneurs how to go from a first prompt to a shipped AI project using Claude Code — bridging the gap between scattered tutorials and production-ready applications.

## Key Features

- **Authentication & subscriptions** — secure accounts with both monthly subscriptions and a lifetime payment option.
- **Content library** — interactive ebooks, ready-to-use skills and step-by-step guides, all in one place.
- **Private community** — a members-only space to ask questions and share progress.
- **Built and operated solo** — product, content, infrastructure and growth.

## Content Produced

I produced the entire catalogue myself:

- **2 interactive ebooks**
- **26 ready-to-use Claude Code skills**
- **29 step-by-step guides**

Together they have reached **2,400+ downloads**.

## Technologies Used

- **Next.js**, **Node.js**, **TypeScript**
- **Supabase** (database, auth)
- **Vercel** (deployment & hosting)

## Outcome

The platform is live at [prompt-academy.fr](https://prompt-academy.fr), serving a growing French-speaking community of developers and entrepreneurs building with applied AI.
